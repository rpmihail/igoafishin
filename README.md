# Go A Fishin’ 2Day Charters — Simple Website

## What you get
A clean, single-page website with:
- Cover photo hero section
- About section
- Pricing (editable in one place)
- Booking/contact links (call/text/email)
- Photo gallery

## Edit the important stuff
Open `site.config.js` and change:
- Email, phone numbers
- Prices
- Gallery photo filenames and captions
- Optional Google Maps link

## Add your photos
Replace:
- `assets/cover.jpg` (your best “hero” photo)
- `assets/photo1.jpg` ... `assets/photo6.jpg`

## Run locally
Just open `index.html` in a browser.

If you want a tiny local server (recommended for testing images):
- Python: `python3 -m http.server 8000`
Then open http://localhost:8000

## Deploy (easy options)
- Drag/drop the folder to Netlify
- Or use GitHub Pages
